Connection:
1. For Arduino

Two_Channel_Relay ------ arduino uno
       VCC ------------------ 5V
       GND ------------------ GND
       IN1 ------------------ D2
       IN2 ------------------ D3

2. For Raspberry Pi

Two_Channel_Relay ------ Raspberry Pi
       VCC ------------------ 5V
       GND ------------------ GND
       IN1 ------------------ GPIO17
       IN2 ------------------ GPIO18
